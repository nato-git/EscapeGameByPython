# 家に帰りたい。

---

## あらすじ

目が覚めると、そこは見覚えのない家の中だった。主人公はこの知らない家から脱出し、自分の家に帰ることができるのか...。

---

## 操作方法

- WASD キーで移動
- E キーで調べる
- SPACE キーでゲーム開始、次のテキストへ
- Q キーで所持品の確認

## 裏話

pygame で初めて制作しているのですが、画像の読み込みや画面の更新などがある分、Unity よりもコードが長くなってしまいますね...<br>
イラストについては、最初すべて自分が描くつもりだったのですが時間的な問題などがあり、泣く泣く AI にお願いしました。<br>
正直マップをもっと大きくしたかったのですが、家の構図をしっかり考えるまではいけませんでした。時間や技術力の問題ではなく、ただの想像力不足です。
